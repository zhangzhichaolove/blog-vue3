# 博客系统-开发中

### 项目架构 vue3 + vue-router + vuex + typescript + element-plus 

### 运行项目
#### 安装依赖
```
npm install
```

#### 运行服务
```
npm run serve
```

### 编译项目
```
npm run build
```

### 错误检查
```
npm run lint
```
